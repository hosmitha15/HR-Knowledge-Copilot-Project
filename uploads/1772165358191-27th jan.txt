Request methods
GET - retrieve information
POST - creates new information/ send some information
PUT - update a record (replace entire obj)
PATCH - update fields in a record (replace a piece of object)
DELETE - Delete a record
OPTIONS - Am I allowed to use this METHOD?


HTTP response status codes - informational, successful, redirection, client error, server error, browser
